=== dydownload 抖音无水印视频下载工具 使用说明 ===

【给普通用户的简易方式】

1. 双击运行 dist/dydownload.exe，出现窗口后服务自动启动

2. 在浏览器中打开抖音 (douyin.com) 并登录

3. 安装浏览器扩展（给 Chrome/Edge 浏览器）：
   - 地址栏输入 chrome://extensions，打开「开发者模式」
   - 点击「加载已解压的扩展」，选择 extension/ 目录

4. 点击浏览器工具栏的 dydownload 插件图标，点「推送到 CLI」
   （之后每 5 分钟会自动推送，不需要重复操作）

5. 下载视频（两种方式任选）：
   a) 浏览抖音视频 → 点插件图标 → 地址已自动填好 → 点「下载无水印视频」
   b) 复制视频链接 → 粘贴到 dydownload.exe 窗口的输入框 → 点「下载」

6. 下载的视频保存在：用户目录/Downloads/dydownload/

【给开发者的命令行方式】

环境要求：conda 环境 dydownload (Python 3.10)

启动后端服务：
  conda run -n dydownload python -m dydownload.cli serve

下载单个视频：
  conda run -n dydownload python -m dydownload.cli download "视频链接"

查看 Cookie 状态：
  conda run -n dydownload python -m dydownload.cli status

测试 a_bogus 签名：
  conda run -n dydownload python -m dydownload.cli test "视频链接"

重新打包 exe（修改源码后）：
  conda run -n dydownload pyinstaller dydownload.spec --distpath ./dist --workpath ./build --noconfirm

【常见问题】

Q: 下载提示 Cookie 过期？
A: 浏览器打开 douyin.com 重新登录，点插件图标推送 Cookie。

Q: 下载提示视频不可用？
A: 视频可能已删除或设为私密，换一个视频试试。

Q: 插件显示「CLI 未启动」？
A: dydownload.exe 还没打开，或已关闭，重新双击运行即可。

Q: 文件保存在哪？
A: C:\Users\你的用户名\Downloads\dydownload\
